Physical-media icons for Cartridge display mode.

SOURCE
  https://github.com/anthonycaccese/xmb-menu-es-de/tree/main/_inc/systems/physical-media

IMPORT INSTRUCTIONS
  Copy the PNG files from the repo's _inc/systems/physical-media/ folder
  directly into this directory, keeping the original filenames.

  Example files (copy as-is):
    nes.png, snes.png, n64.png, gb.png, gbc.png, gba.png
    psx.png, ps2.png, psp.png, psvita.png
    megadrive.png, genesis.png, mastersystem.png, gamegear.png
    saturn.png, dreamcast.png
    gc.png, wii.png, wiiu.png
    nds.png, n3ds.png
    pcengine.png, neogeo.png, ngp.png
    atari2600.png, atari7800.png, atarilynx.png
    virtualboy.png, wonderswan.png, wonderswancolor.png
    switch.png, c64.png, amiga.png, mame.png, arcade.png

FALLBACK
  If a PNG is missing for a platform, the launcher automatically falls back
  to the built-in generic media shape (disc, cartridge, UMD, etc.).
